Add to Forage

Drag this button to your bookmarks bar:

+ Forage

Then click it on any book page to add it to your library.

How it works

  1. Click the bookmarklet on any book page
  2. It scrapes the title and author from the page
  3. Opens Forage with the add form pre-filled
  4. Auto-triggers an Open Library lookup for extra metadata
  5. Review, edit, and save

Supported sites

Works on any site with standard metadata. Tested with:

  • Amazon
  • Goodreads
  • Bookshop.org
  • Open Library
  • Any site with og:title or JSON-LD Book data
